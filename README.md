# HyprDisJust

HyprDisJust is a Hyprland monitor profile manager. The idea is to save monitor layouts and re-apply them automatically when the same monitors are connected again with hot-plug and automatic save.

## Status

Very early. Right now this is mostly notes and direction, not a usable tool yet.

Aiming for a small Rust CLI first. Anything fancier can come later if the basic workflow proves useful.

## Goals

- Save the current Hyprland monitor layout
- Match monitors by something stable like description
- Re-apply a saved layout when that monitor setup appears again
- Get functioning hot-plug
